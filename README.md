# NET Athlete

NET Athlete est une application web qui connecte les athlètes, clubs, recruteurs, agents et sponsors.

## Structure du projet

- frontend/ : interface utilisateur
- backend/ : logique serveur et API
- database/ : structure de la base de données
- docs/ : documentation du projet
- prompts/ : prompts utilisés avec les IA

## Branches

- main : version stable
- dev : version en développement
- feature/auth : inscription et connexion
- feature/profile : profils utilisateurs
- feature/search : recherche de talents
- feature/messages : messagerie
